For the detection and Prevention of the disease of plants from getting spread this Project Is Developed. In this project, image analysis is done using a machine learning algorithm. It has many advantages for the use in big farms of crops and thus it automatically detects signs of disease whenever they appear on leaves of the plant. The term disease means the type of damage to the plants. This project the best method for detection of plant diseases, using image processing and alerting about the disease caused by sending email, SMS or sending a volunteer and displaying the name of the disease, giving the user information regarding proper medications to the crop and locations to access the pesticides or insecticides. To upgrade agricultural products, automatic detection of disease symptoms is useful.
